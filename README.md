# Fork of Wanikani font randomizer
Author: Rufus Kogg Röjder

The original purpose of this fork was to add the functionality to treat quizzes during lessons differently from the “recent mistakes” mode. In the upstream version, these are the same since they are both labelled quiz in the URL.

The purpose has been extended to adding an extended feature set to the extension. The desired features are
- [ ] The ability to select which fonts are in the pool of fonts 
- [ ] Allowing for randomizing the font on each question instead of just on each session.
- [ ] Allowing one to quickly toggle back to the base font in case the font is hard to read.
- [ ] Allowing the user to customize the "base" font used in cases where randomization is disabled as well as in lessons.
- [ ] Adapting the general styling of all UI elements.
